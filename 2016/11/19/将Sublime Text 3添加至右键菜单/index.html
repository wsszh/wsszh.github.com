将Sublime Text 3添加至右键菜单

发表于 2016-11-19   |   分类于 文本编辑器   |     |   阅读次数

想找一款好用的Python的编辑器，网上很多人推荐Sublime Text 3，因此入坑，但我下载的是绿色汉化破解版，直接解压就可以运行，如果想要右键用Sublime Text 3打开文本文件，就必须自己进行系统设置，步骤如下：

我的操作系统是Win10

打开注册表

Win + R打开运行窗口，输入regedit，回车：

查找节点

在注册表编辑器中找到 /计算机/HKEY_CLASSES_ROOT/*/shell

新建节点

在 /计算机/HKEY_CLASSES_ROOT/*/shell 中新建项命名为 Edit with Sublime Text 3

在新建项中新建字符串值，命名为 Icon，右键Icon，点击“修改”，将数值数据修改为 D:\software\SublimeText3_ha\Sublime Text 3.3126x64\sublime_text.exe,0 （逗号前为系统中Sublime Text 3的exe文件的路径）

然后在新建项中新建项命名为 Command （必须为这个名称），修改右侧窗口“（默认）”的数据为 D:\software\SublimeText3_ha\Sublime Text 3.3126x64\sublime_text.exe %1
这里要注意 % 和 1 之间不能有空格

检验结果

选择一个txt文件，点击右键，菜单中会出现：
